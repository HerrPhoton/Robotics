1) alias - Создает псевдонимы для команд.
2) history - Отображает все недавно введённые команды.
3) whatis - Отображает краткое описание какой‑либо программы.
4) htop - Отображает список процессов с использованием графиков и позволяет легко управлять ими.
5) file - Выводит информацию об указанном файле.
6) df - Отображает объём вашего диска и количество свободного места на нём. 
7) uptime - Сообщает, как давно запущена система.
8) xkill - команда для завершения процессов. После ввода необходимо кликнуть по окну, которое нужно закрыть.
9) grep - Поиск текстовых файлов, содержащих определённые строки.
10) lsblk - Демонстрирует, какие диски есть в вашей системе и на какие разделы они поделены.
