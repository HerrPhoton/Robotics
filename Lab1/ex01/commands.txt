1) alias - Создает псевдонимы для команд: alias исходная_команда псевдоним.
2) history - Отображает все недавно введённые команды.
3) whatis - Отображает краткое описание какой‑либо программы: whatis имя_пакета.
4) sudo gksudo - Запуска с правами администратора приложения с графическим интерфейсом.
5) sudo apt‑add‑repository адрес_репозитория - Добавить сторонний репозиторий.
6) sudo apt purge имя_пакета - Удалить ненужный пакет со всеми зависимостями.
7) sudo apt autoremove - Удалить все ненужные зависимости, бесхозные пакеты и т.д.
8) xkill - команда для завершения процессов. После ввода необходимо кликнуть по окну, которое нужно закрыть.
9) grep - Поиск текстовых файлов, содержащих определённые строки.
10) lsblk - Демонстрирует, какие диски есть в вашей системе и на какие разделы они поделены.
